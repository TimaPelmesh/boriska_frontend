Основы HTML

HTML (HyperText Markup Language) — язык разметки гипертекста, стандартный язык для создания веб-страниц. HTML описывает структуру веб-страницы с помощью элементов, которые сообщают браузеру, как отображать содержимое.

Что такое HTML?

HTML — это не язык программирования, а язык разметки. Он использует специальные теги для описания структуры и содержания веб-страницы.

Основные характеристики HTML:

  • Определяет структуру веб-страницы
  • Состоит из элементов (тегов)
  • Не имеет логики программирования
  • Интерпретируется браузером

Структура HTML-документа

Каждый HTML-документ имеет базовую структуру:

<!DOCTYPE html>
<html>
<head>
    <title>Заголовок страницы</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это мой первый HTML-документ.</p>
</body>
</html>

Давайте разберем каждый элемент:

  • <!DOCTYPE html> — объявление типа документа HTML5
  • <html> — корневой элемент веб-страницы
  • <head> — содержит метаданные и ссылки на внешние файлы
  • <title> — определяет заголовок страницы (отображается на вкладке браузера)
  • <body> — содержит все видимое содержимое страницы
  • <h1> — заголовок первого уровня
  • <p> — параграф (абзац)

HTML-элементы и теги

HTML-элементы обозначаются тегами, записываемыми в угловых скобках:

<tagname>Содержимое...</tagname>

Элементы HTML состоят из:

  • Открывающего тега — <tagname>
  • Содержимого — текст или другие элементы
  • Закрывающего тега — </tagname>

Некоторые элементы не имеют содержимого и закрывающего тега — это пустые элементы, например:

<img src="image.jpg" alt="Описание изображения">

Атрибуты HTML

Атрибуты предоставляют дополнительную информацию об элементах:

<tagname attribute="value">Содержимое...</tagname>

Основные атрибуты HTML:

  • id — уникальный идентификатор элемента
  • class — класс для применения CSS-стилей
  • style — встроенные CSS-стили
  • src — источник для элементов мультимедиа
  • href — URL для ссылок
  • alt — альтернативный текст для изображений

Практическое задание

Создайте простую HTML-страницу со следующей структурой:

  1. Заголовок страницы "Моя первая HTML-страница"
  2. Заголовок первого уровня с вашим именем
  3. Параграф с кратким описанием о себе
  4. Список ваших увлечений

Подсказка:

<!DOCTYPE html>
<html>
<head>
    <title>Моя первая HTML-страница</title>
</head>
<body>
    <h1>Ваше имя</h1>
    <p>Краткая информация о себе...</p>
    <h2>Мои увлечения:</h2>
    <ul>
        <li>Увлечение 1</li>
        <li>Увлечение 2</li>
        <li>Увлечение 3</li>
    </ul>
</body>
</html>
