Основы CSS

CSS (Cascading Style Sheets) — каскадные таблицы стилей, используются для оформления веб-страниц. CSS позволяет отделить визуальное представление документа от его содержимого (HTML).

Что такое CSS?

CSS — это язык таблиц стилей, который определяет, как элементы HTML отображаются на экране, на бумаге или в других медиа.

Основные преимущества CSS:

  • Разделение содержания и оформления
  • Контроль всех аспектов внешнего вида страницы
  • Повторное использование стилей на многих страницах
  • Адаптивность под разные устройства
  • Повышение скорости загрузки страниц

Синтаксис CSS

CSS-правило состоит из селектора и блока объявлений:

селектор {
    свойство1: значение1;
    свойство2: значение2;
    свойство3: значение3;
}

Например:

h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

Здесь:

  • селектор (h1) указывает на элемент HTML, который нужно стилизовать
  • свойства (color, font-size, text-align) определяют, какие характеристики элемента изменяются
  • значения (blue, 24px, center) задают новые значения для свойств

Способы подключения CSS

Существует три основных способа добавления CSS в HTML-документ:

1. Внешний CSS (рекомендуется)

CSS-код размещается в отдельном файле с расширением .css, который подключается к HTML с помощью тега link:

<head>
    <link rel="stylesheet" href="styles.css">
</head>

2. Внутренний CSS

CSS-код помещается внутри тега style в разделе head:

<head>
    <style>
        body {
            background-color: lightblue;
        }
        h1 {
            color: navy;
        }
    </style>
</head>

3. Встроенный CSS

CSS-код добавляется непосредственно к элементу HTML с помощью атрибута style:

<h1 style="color: blue; text-align: center;">Заголовок</h1>

Основные селекторы CSS

Селекторы определяют, к каким элементам HTML применяются стили:

Селектор Пример Описание
Селектор элемента p { } Выбирает все элементы <p>
Селектор ID #id { } Выбирает элемент с конкретным id
Селектор класса .class { } Выбирает все элементы с указанным классом
Универсальный селектор * { } Выбирает все элементы
Селектор потомков div p { } Выбирает все <p> внутри <div>

Основные свойства CSS

Цвета и фон

  • color — цвет текста
  • background-color — цвет фона
  • background-image — изображение фона

Текст и шрифты

  • font-family — семейство шрифта
  • font-size — размер шрифта
  • font-weight — толщина шрифта
  • text-align — выравнивание текста
  • text-decoration — декорирование текста (подчеркивание и т.д.)

Размеры и отступы

  • width — ширина элемента
  • height — высота элемента
  • margin — внешние отступы
  • padding — внутренние отступы
  • border — граница элемента

Позиционирование

  • position — метод позиционирования
  • display — тип отображения элемента
  • float — обтекание элемента
  • z-index — порядок наложения

Практическое задание

Создайте HTML-страницу с тремя абзацами текста и примените различные стили:

  1. К первому абзацу примените красный цвет текста и выравнивание по центру
  2. Ко второму абзацу — синий цвет текста, курсивное начертание и увеличенный размер шрифта
  3. К третьему абзацу — зеленый цвет текста, жирное начертание и рамку

Пример HTML:

<!DOCTYPE html>
<html>
<head>
    <title>Практика CSS</title>
    <style>
        .paragraph1 {
            color: red;
            text-align: center;
        }
        
        .paragraph2 {
            color: blue;
            font-style: italic;
            font-size: 20px;
        }
        
        .paragraph3 {
            color: green;
            font-weight: bold;
            border: 1px solid black;
            padding: 10px;
        }
    </style>
</head>
<body>
    <p class="paragraph1">Первый абзац с красным цветом и выравниванием по центру.</p>
    <p class="paragraph2">Второй абзац с синим цветом, курсивом и увеличенным шрифтом.</p>
    <p class="paragraph3">Третий абзац с зеленым цветом, жирным шрифтом и рамкой.</p>
</body>
</html>

Наследование и каскадность

CSS имеет два важных принципа:

Наследование

Некоторые свойства CSS передаются от родительских элементов к дочерним. Например, если вы установите color для body, все элементы внутри body унаследуют этот цвет, если для них не задано другое значение.

Каскадность

Определяет, какие стили применяются в случае конфликтов. Порядок приоритета (от высшего к низшему):

  1. Встроенные стили (атрибут style)
  2. Внутренние и внешние стили
  3. Стили браузера по умолчанию

При одинаковом уровне приоритета последний объявленный стиль имеет преимущество.
